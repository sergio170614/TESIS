# Predicción del Monto de Siniestros en Seguros de Automóviles

Este repositorio contiene el código utilizado para la investigación **"Predicción del monto de siniestro en vehículos de los segmentos compactos y subcompactos mediante técnicas de aprendizaje automático en México 2024 a 2025"**.

## Descripción

Se implementaron y compararon tres modelos supervisados de regresión para estimar el monto de siniestros en seguros de automóviles compactos y subcompactos en México:

- Regresión Lineal Múltiple
- Random Forest
- XGBoost (con distribución Tweedie)

## Contenido

- Preprocesamiento y limpieza de datos
- Análisis Exploratorio de Datos (EDA)
- Feature Engineering
- Entrenamiento y evaluación de modelos
- Optimización de hiperparámetros
- Interpretabilidad con SHAP

## Tecnologías

- Python
- pandas, numpy, scikit-learn, xgboost, shap, matplotlib, seaborn

## Datos

Los datos provienen de la base estadística RR-8 de la Comisión Nacional de Seguros y Fianzas (CNSF), correspondiente al año de ejercicio 2024.
